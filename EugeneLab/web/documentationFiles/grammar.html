The Grammar of the miniEugene Language

The provided syntax in this Web-based miniEugene UI is a subset of miniEugene's grammar. 

script                = (cnf)+

cnf                   = constraint ("or" cnf)? LINE_FEED 

constraint            = ("NOT")? 
                        ( ( (ID)? constraint_operator (ID | INT)? ) | 
                            template_constraint )

constraint_operator   = ["contains"|"CONTAINS"| ... |"drives"|"DRIVES"]

template_constraint   = (name)? template_operator list_of_ids

template_operator     = ["template"|"TEMPLATE"|"sequence"|"SEQUENCE"]

list_of_ids           = ( "[" selection "]" | ID ) ("," list_of_ids)?

selection             = ID ( "|" selection )? 

ID                    = [a-zA-Z]+

INT                   = [0-9]+
